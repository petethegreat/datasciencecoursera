#Some markdown stuff#

- wrapping things in "\#" makes headings '''#heading#'''

- backslashes (\\) to escape special markdown characters
- links: \[link text\]\(url\) - cheatsheet located [here](https://enterprise.github.com/downloads/en/markdown-cheatsheet.pdf)
